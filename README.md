# WebDesk Client

Aplicativo de desktop para conexão com um servidor web

## Instalação

```shell
git clone https://github.com/dxsuporte/webdesk-client
```

```shell
cd webdesk-client
```

```shell
npm install
```

## Compilação

```shell
npm run build
```

Linux: _`npm run build-linux`_ Windows: _`npm run build-win`_
